# 🎤 KaraSync Studio

> Professional Karaoke Video Studio powered by React, TypeScript, FFmpeg and AI.

KaraSync Studio is a desktop application designed to create professional karaoke videos with automatic synchronization, advanced subtitle styling, and high-quality video export.

---

## ✨ Features

### 🎬 Video
- Import MP4, MOV, AVI...
- Video preview
- FFmpeg integration

### 📝 Subtitles
- Import SRT
- Import ASS
- Import TXT
- Import CSV
- Import JSON

### 🤖 AI Synchronization (Coming Soon)
- Automatic word alignment
- ElevenLabs integration
- Word-by-word timing
- Automatic timeline generation

### 🎨 Karaoke Styles

Included presets:

- 🎵 YouTube Kids
- 🌈 Cocomelon
- 🎬 Disney
- 🍎 Apple Music
- 📺 Netflix
- 🎧 TikTok

Customizable:

- Font
- Size
- Colors
- Outline
- Shadow
- Vertical Position
- Animation
- Highlight Style

## 🚀 Installation

```bash
git clone https://github.com/mendilinassima-glitch/karaoke-video-app.git

cd karaoke-video-app

npm install

npm run dev
```
